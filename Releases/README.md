# Releases

Welcome to the releases page for **Falculator**, a gamified financial simulator inspired by games like Dwarf Fortress, Plague Inc., and Space Trader.

# About Releases

Release builds for Falculator are available here. Currently, only **Beta Releases** are available, which have a built-in expiry date to encourage feedback and updates.
